# 📚 Book Scraper Project

مشروع بسيط لعمل **Web Scraping** لموقع كتب، ثم تنظيف البيانات وتنظيمها باستخدام Python.

---

## 🎯 الهدف من المشروع

هذا المشروع يقوم بـ:

- سحب بيانات الكتب من موقع:
  http://books.toscrape.com

- استخراج:
  - اسم الكتاب
  - السعر
  - التقييم
  - صورة الكتاب

- حفظ البيانات الخام (Raw Data)

- تنظيف البيانات وتحويلها

- تقسيم البيانات حسب التقييم (1 إلى 5)

- تنظيم الصور داخل مجلدات حسب التقييم

---

## 🧠 التقنيات المستخدمة

- Python 🐍
- requests
- BeautifulSoup
- pandas
- pathlib
- shutil
