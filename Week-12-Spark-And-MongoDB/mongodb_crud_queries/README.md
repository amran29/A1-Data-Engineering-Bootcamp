# MongoDB Assignment

## Overview

This project contains solutions for a MongoDB assignment implemented using **Python** and **PyMongo**. The notebook demonstrates essential MongoDB operations including database creation, document insertion, querying, updating, deleting, sorting, and aggregation.

The objective of this assignment is to provide practical experience in working with MongoDB from Python and understanding the fundamental CRUD operations used in NoSQL databases.

---

## Technologies Used

* MongoDB
* Python 3.x
* PyMongo
* Jupyter Notebook

---

## Assignment Tasks

The notebook covers the following operations:

1. Connect to MongoDB server.
2. Create a database.
3. Create a collection.
4. Insert one document.
5. Insert multiple documents.
6. Retrieve documents using queries.
7. Update existing documents.
8. Delete documents.
9. Sort and filter records.
10. Perform aggregation operations.

## Installation

Install the required package:

```bash
pip install pymongo
```

Make sure MongoDB is installed and running on your local machine.

---

## Running the Project

1. Start MongoDB service.
2. Open Jupyter Notebook.
3. Open:

```text
mongodb_assignment_solved.ipynb
```

4. Execute the notebook cells sequentially.
